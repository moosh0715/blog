# SIGNAL / NOTES

레이다·DSP·FPGA 개인 기술 블로그. GitHub Pages와 Jekyll로 동작합니다.

## 새 글 올리기

1. `_posts` 폴더에 `YYYY-MM-DD-영문-주소.md` 파일을 만듭니다.
2. 아래 front matter를 넣고 Markdown으로 작성합니다.
3. main 브랜치에 저장하면 GitHub Pages가 자동 배포됩니다.

```md
---
layout: post
title: "글 제목"
date: 2026-07-23
description: "목록에 표시할 설명"
tags: [RADAR, DSP]
---

## 첫 번째 제목

본문을 Markdown으로 작성합니다.
```

배포 주소: https://moosh0715.github.io/blog/
